# 1. Visão Geral do Produto

O **CRM Solar** é uma plataforma completa para gestão de leads e vendas, desenvolvida especificamente para empresas do setor de energia solar.

## 1.1 Funcionalidades Principais

| Módulo | Descrição |
|--------|-----------|
| Chatbot WhatsApp | Atendimento automático 24/7 com IA |
| Gestão de Leads | Captura, qualificação e scoring automático |
| Kanban de Vendas | Funil visual com 5 etapas personalizáveis |
| Propostas | Geração automática de orçamentos |
| Dashboard | KPIs em tempo real por vendedor |
| Multi-tenant | Isolamento total de dados por empresa |
